# Support Guide

Need help with the AIUB Course Recommendation System? This document explains where to go depending on the type of issue.

## Quick Answers

- Common questions: See `FAQ.md`
- Planned features: See `ROADMAP.md`
- How to contribute: See `CONTRIBUTING.md`

## Channels

| Type             | Where                 | Notes                        |
| ---------------- | --------------------- | ---------------------------- |
| Bug Report       | GitHub Issues         | Include steps & browser info |
| Feature Request  | GitHub Issues         | Tag with `enhancement`       |
| Security Concern | See `SECURITY.md`     | Private disclosure preferred |
| General Question | Issues or Discussions | Mark as `question`           |

## Service Levels (Informal)

| Severity | Example                    | Target Initial Response |
| -------- | -------------------------- | ----------------------- |
| Critical | Broken core UI, JSON crash | 48h                     |
| Normal   | Minor display bug          | 5 days                  |
| Low      | Typos, minor layout tweak  | 7 days                  |

(These are best-effort targets; this is a volunteer-maintained project.)

## Before Submitting a Bug

1. Pull latest `main`
2. Hard refresh (Ctrl+F5)
3. Reproduce in another browser
4. Check console (F12) for errors
5. Confirm JSON file validity

## Provide Useful Info

Please include:

- Summary
- Steps to reproduce
- Expected vs actual
- Screenshots (if visual)
- Browser + version
- Any console errors (copy text)

## No Direct Support?

If real-time chat is desired, consider opening a discussion to gauge interest in a community channel (e.g., Discord or Gitter) once user base grows.
